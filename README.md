# Background-Remover-by-Zee
==============================
A Python-based background removal tool using deep learning to automatically remove image backgrounds.
-------------------
## Description
========================
This project removes the background from images and outputs a transparent PNG file. It is designed as a simple command-line tool for academic learning and experimentation.
------------------------------
## Features
=======================
- Automatic background removal
- Supports JPG and PNG input
- Outputs transparent PNG images
- Simple command-line usage
------------------------
## Technologies Used
===================
- Python
- rembg
- Pillow (PIL)
